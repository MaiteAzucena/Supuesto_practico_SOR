## 2. Integración del cliente Linux Mint Cinnamon al dominio AD DS
- Pasos:
- 1. Configurar las características del equipo.
- 2. Instalar el software necesario.
- 3. Configurar Kerberos.
- 4. Comprobar el funcionamiento de Kerberos.
- 5. Unir el cliente al dominio.
- 6. Configurar SSSD.
- 7. Ajustar el comportamiento de PAM para iniciar sesión con usuarios del dominio.
- 8. Convertir al administrador del dominio en administrador local.
- 9. Reiniciar el equipo y comprobar el inicio de sesión gráfico.
- 10. Iniciar sesión con cualquier cuenta del dominio.
